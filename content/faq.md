**What is this blog?**

Learning notes on machine learning, mathematics, and related topics — written for my own understanding, shared in public.

**Languages?**

Each post is written in either English or 中文, whichever I prefer for that topic. There is no requirement to publish both versions.

**Math rendering?**

Formulas use [KaTeX](https://katex.org/). If something looks wrong, please open an issue on GitHub.

**How often do you post?**

No fixed schedule — whenever I finish a topic worth writing up.
